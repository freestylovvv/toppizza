# 🎨 TopPizza - Обзор функционала

## Главная страница

### Шапка сайта
- **Логотип TopPizza** - яркий оранжевый (#ff6900)
- **Город и время доставки** - "Тюмень, 30-40 мин"
- **Кнопка входа** - с hover эффектом

### Баннер
- Градиентный фон (оранжевый)
- Крупный заголовок "Вкусная пицца с доставкой"
- Подзаголовок "Доставим за 30-40 минут или вернем деньги"

### Навигация по категориям
- Sticky меню с категориями
- Плавная прокрутка к разделам
- Hover эффекты (белый → оранжевый)

## Цветовая схема

- **Основной:** #ff6900 (оранжевый)
- **Hover:** #ff7a1a (светлее)
- **Текст:** #191919 (почти черный)
- **Вторичный текст:** #a5a5a5 (серый)
- **Фон:** #f9f9f9 (светло-серый)
- **Белый:** #fff

## Анимации

- **Hover карточек:** translateY(-4px) + shadow
- **Кнопки:** изменение цвета фона
- **Переключатели размера:** плавное изменение фона и тени
- **Модальное окно:** fade in/out

## Адаптивность

- Desktop: 1320px max-width
- Карточки: grid с auto-fill (минимум 290px)
- Корзина: фиксированная ширина 380px
- Sticky элементы: header и навигация

## База данных

### Таблицы

**Category**
- id, name

**Product**
- id, name, imageUrl, categoryId

**Variant**
- id, productId, size, price

**Order**
- id, fullName, email, phone, address, totalPrice, status, createdAt

**OrderItem**
- id, orderId, productId, variantId, quantity, price

- Мясная (495-895₽)
- Гавайская (425-825₽)
- Барбекю (475-875₽)

**Закуски (4 шт):**
- Куриные крылышки (295₽)
- Картофель фри (145₽)
- Наггетсы (245₽)
- Сырные палочки (195₽)

**Напитки (2 шт):**
- Coca-Cola (95₽)
- Сок апельсиновый (115₽)

## API Endpoints

### POST /api/orders
Создание нового заказа

**Request:**
```json
{
  "fullName": "Иван Иванов",
  "email": "ivan@example.com",
  "phone": "+7 999 123-45-67",
  "address": "ул. Ленина, д. 1, кв. 1",
  "items": [
    {
      "productId": 1,
      "variantId": 2,
      "quantity": 2,
      "price": 595
    }
  ],
  "totalPrice": 1190
}
```

**Response:**
```json
{
  "success": true,
  "orderId": 1
}
```
