# 🍕 TopPizza - Инструкция по запуску

## Что уже сделано

✅ Современный дизайн в стиле Додо Пицца
✅ Адаптивная верстка
✅ Корзина с localStorage
✅ Оформление заказов
✅ База данных PostgreSQL с Prisma
✅ API для создания заказов

## Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка базы данных

Убедитесь, что PostgreSQL запущен. База данных `toppizza` должна существовать.

Если нужно создать базу:
```bash
createdb toppizza
```

### 3. Инициализация Prisma

```bash
npm run prisma:generate
npm run prisma:push
npm run prisma:seed
```

### 4. Запуск проекта

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Основные улучшения

### Дизайн
- 🎨 Современная цветовая схема (#ff6900 - оранжевый)
- 🎯 Плавные анимации и переходы
- 📱 Адаптивная верстка
- 🖼️ Реальные изображения из Unsplash

### Функционал
- 🛒 Корзина с подсчетом товаров
- 📦 Выбор размера пиццы
- 💰 Автоматический подсчет суммы
- 📝 Форма оформления заказа
- 🔄 Обновление корзины в реальном времени

### Компоненты

#### Header
- Логотип TopPizza
- Город и время доставки
- Кнопка входа

#### ProductCard
- Изображение товара
- Название и описание
- Выбор размера (для пицц)
- Кнопка добавления в корзину
- Hover эффекты

#### Cart
- Список товаров
- Изменение количества
- Подсчет итоговой суммы
- Модальное окно оформления заказа

### База данных

Структура:
- **Category** - категории (Пицца, Закуски, Напитки)
- **Product** - товары
- **Variant** - варианты (размеры и цены)
- **Order** - заказы
- **OrderItem** - позиции в заказе

## Технологии

- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - строгая типизация
- **PostgreSQL** - реляционная база данных
- **Prisma** - современная ORM
- **CSS-in-JS** - inline стили для быстрой разработки

## Поддержка

Если возникли проблемы:

1. Проверьте, что PostgreSQL запущен
2. Убедитесь, что DATABASE_URL в .env правильный
3. Попробуйте пересоздать базу данных
4. Проверьте версии Node.js (рекомендуется 18+)
